Install Steps
To run DnDNotetaker, you will need Visual Studio 2026 installed and set up.
It can be installed here.
https://visualstudio.microsoft.com/downloads/

To install the DnD notetaker app, you should click the green Code button and press install zip.

After the zip is installed, extract it by right-clicking on it and pressing "extract all."

After this, press ok, and open the newly created, unzipped folder.

Locate the D&DNoteTaker.slnx file and double-click it to open the project in Visual Studio.

After Visual Studio launches, look at the top of your screen and find a small green play button. Press the button and the website will launch in your default browser.

No environment variables are necessary.

This website is not deployed.

Technical Information:

This software uses MongoDB Atlas as a backend database, Blazor ASP.NET Core as a frontend, and has some steps for deployment to complete on Microsoft Azure.
